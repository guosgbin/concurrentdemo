###notuse包
没有用Single Threaded Execution模式的例子
- Gate类：表示 门 类，人通过时会将人的姓名和地址打印出来。
- People类：表示一个 人 的线程。
- Main类： 测试类。

###use包
使用Single Threaded Execution模式的例子
- Gate类：表示 门 类，人通过时会将人的姓名和地址打印出来。
- People类：表示一个 人 的线程。
- Main类： 测试类。

###problem包
多个SharedResource共享资源带来的问题

- Chopsticks类：筷子类，吃饭需要两双筷子
- People类：表示一个 人 的线程。
- Main类： 测试类。

###solveproblem包
解决多个SharedResource共享资源带来的问题

- Chopsticks类：筷子类，吃饭需要两双筷子
- Pair类：封装两个筷子类
- People类：表示一个 人 的线程。
- Main类： 测试类。
